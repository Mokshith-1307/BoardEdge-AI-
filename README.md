# BoardEdge AI – Intelligent Academic Performance Analyzer

BoardEdge AI is a web-based academic analytics platform designed to help students track performance, predict academic risk, and estimate selection probability.

## 🚀 Features
- Secure Login & Registration
- Risk Level Prediction (High / Medium / Low)
- Selection Probability Estimation
- Performance History Tracking
- Leaderboard Ranking
- Admin Analytics Dashboard

## 🛠 Technologies Used
- Python (Flask)
- SQLite
- HTML/CSS
- Chart.js

## ⚙️ Installation

1. Clone the repository
2. Install dependencies:
   pip install flask
3. Run the application:
   python app.py
